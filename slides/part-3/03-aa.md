## Resources

credentials.yml
```
GIT_REPO: https://github.com/cah-JimShingler/spring-music.git
GIT_BRANCH: master
```

pipeline.yml
```
resources:
  - name: source-code
    type: git
    source:
      uri: {{GIT_REPO}}
      branch: {{GIT_BRANCH}}
. . .

jobs:
  - name: unit-tests
    plan:
    - get: source-code
      trigger: true
```
