## Tasks

What is a task?
<br>
A task is the smallest "function" that handles input, processes and sends outputs.
<br>
```
jobs:
  - name: unit-tests
    plan:
    - get: source-code
      trigger: true
    - task: run-tests
      privileged: true
      file: source-code/ci/tasks/unit.yml
      params:
        SONAR: {{SONAR}}
```
