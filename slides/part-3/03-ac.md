## Use Custom Resources

pipeline.yml

```
job:
...
- name: package-publish
  plan:
  ...
  - put: build-artifact-jar
    params:
      file: build/spring-music.war
      pom_file: source-code/pom.xml
...
- name: deploy-to-dev
  plan:
  - get: source-code
    passed: [ package-publish]
    trigger: true
  - get: build-artifact-jar
```
