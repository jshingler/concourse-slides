## Resources - Custom Definition

pipeline.yml
```
resource_types:
  - name: maven-resource
    type: docker-image
    source:
      repository: patrickcrocker/maven-resource
      tag: latest
...
- name: build-artifact-jar
    type: maven-resource
    source:
      url: {{MAVEN_REPO}}
      artifact: com.example:spring-music:war
      username: {{MAVEN_USERNAME}}
      password: {{MAVEN_PASSWORD}}      
```
