## Lab 1 -- Setup

### Clone the workshops

```
git clone https://github.com/jshingler/CI-Stack.git
cd CI-Stack
```

See the [CI-Stack Readme](https://github.com/jshingler/CI-Stack) for key setup instructions

<br>
### Start the CI / CD Pipeline

```
docker-compose up
```
